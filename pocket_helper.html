Question Cards/Studying Prep

PocketHelper is a plataform to help students studying for IT exams to have quick question cards on the go without the overhead of signing up or registering.

It was created by having a react front end that fetches data (the questions) from the backend which is provided by the Django Rest Framework.

Technologies:
  • - Python
  • - Django + Django REST Framework
  • - React
  • - Heroku
